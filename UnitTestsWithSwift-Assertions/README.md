### XCTAssertTrue
- Asserts that an expression is true.
### XCTAssertFalse
- Asserts that an expression is false.
### XCTAssertNil
- Asserts that an expression is nil.
### XCTAssertNotNil
- Asserts that an expression is not nil.
### XCTAssertEqual
- Asserts that two values are equal.

### Technologies Used
